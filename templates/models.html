1st Machine Learning Model Decision Tree

Description

A Decision Trees are useful machine learning tools for making decisions and categorizing data. They create a tree-like strucutre by dividing data based on important features, forming decision nodes and leaf nodes. The algorithm keeps splitting the data until it reaches a stopping point, resulting in easy-to-understand models that handle different types of data. Decision trees find applications in various fields and are great for solving classification and regression problems by capturing complex relationships in the data

2nd Machine Learning Model Random Forest

Description

A Random Forest is a popular machine learning model that belongs to the ensemble learning family. It is used for both classification and regression tasks. The fundamental idea behind a Random Forest is to combine the predictions of multiple individual decision trees to create a more accurate and robust overall prediction.

3rd Machine Learning Model Stochastic Gradient Descent

Description

Stochastic Gradient Descent (SGD) is a fundamental optimization algorithm widely used in machine learning and neural network training. Unlike conventional gradient descent, which processes the entire dataset in each iteration, SGD takes a more computationally efficient approach by randomly sampling individual data points or small batches for updating model parameters. This randomness introduces noise, causing the optimization path to oscillate around the true gradient, but this trade-off enables faster convergence, especially on large datasets. By updating the model parameters based on the gradients of the selected data points, SGD continually adapts and refines the model, eventually finding the optimal solution. Additionally, variants like Mini-batch SGD strike a balance between computation and convergence speed, making SGD a versatile and widely adopted optimization technique in the field of machine learning.

4th Machine Learning Model Multi Layered Perceptron Neural Network

Description

A neural network is made up of many nodes and layers. Each node has weights associated to it that determines the activation function parameters and threshold. An activation function, such as the ReLU function, takes an input and based on that, gives an output which is above or below the activation threshold, thus causing the node to be activated or disabled. Each layer has some number of nodes, with all the nodes feeding into the input of the next layer for this particular model type. Finally, an output is generated from all the interactions of the different nodes

Limitations

Some models, such as the Multi Layer Perceptron neural network, overfit to the training data and had poor validation scores. The overfitting can likely be attributed to the small dataset size, although most models better handled the size of the data. Most of the SKLearn models performed very well due to a combination of hyperparameter tuning, feature engineering, and data scaling. The neural network, our only model using Pytorch instead of SKlearn, was not highly tuned due to time limitations.

The size of our dataset is an obvious limitation; however, our accuracy was high for most models. Time was a limiting factor as well, specifically limiting how long we could spend tuning our neural network. Another challenge was trying to find the most important features in the dataset, as there were a lot of features that improved the model performance, but were obscure and not usable if the model was deployed for public use.
